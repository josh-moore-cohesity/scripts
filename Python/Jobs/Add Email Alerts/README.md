
# **append-vmware-pg-alert-recipients.py**

   Append alert recipients for all VMWare PGs on specified clusters.

## **Examples**

    python append-vmware-pg-alert-recipients.py -i -ea user1@domain.com -ea user2@domain.com
    
## **Download**

    curl -O https://raw.githubusercontent.com/josh-moore-cohesity/scripts/refs/heads/main/Python/Jobs/Add%20Email%20Alerts/append-vmware-pg-alert-recipients.py
    curl -O https://raw.githubusercontent.com/cohesity/community-automation-samples/main/python/pyhesity.py

[pyhesity.py](https://github.com/bseltz-cohesity/scripts/tree/master/python/pyhesity) is required
