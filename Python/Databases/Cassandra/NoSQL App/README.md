# **nosql_app_status.py**

   Get Status of the NoSQL App and Version.<br />
   [pyhesity.py](https://github.com/bseltz-cohesity/scripts/tree/master/python/pyhesity) is required 
   
   
## **Examples**

    python nosql_app_status.py -i
    python nosql_app_status.py -i -c clusterA ClusterB
    python nosql_app_status.py -i -cl clusterlist.txt
